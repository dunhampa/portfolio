Project #2: Beauty in a Bug

Summary
Using Processing.org (the programming language), I built all the graphics, circles, colors, and animations for this site. The best part of using a programming language for graphical work is that there is no such thing as a ugly bug. Similar to when Bob Ross makes a "happy little tree" out of a accidental brush stroke -- in my first attempt to create the website banner (circle array at top of page), I made a mistake in incrementing my loop counter. The output was a stacked colorful facet, instead of an organized array of circles.

Technologies: Processing.org, Java
